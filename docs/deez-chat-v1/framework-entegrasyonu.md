# Framework Entegrasyonu

Deez Chat V1, ESX ve QBCore sunucularında kullanılmak üzere yapılandırılabilir.

## ESX

`config.lua` içerisinde framework değerini ESX olarak seçin ve ESX kaynağının Deez Chat'ten önce başladığından emin olun.

## QBCore

Framework değerini QBCore olarak seçin ve `qb-core` kaynağının Deez Chat'ten önce başladığından emin olun.

Özel framework değişikliklerinde Discord destek kanalından yardım alabilirsiniz.
