# Framework Integration

Deez Chat V1 can be configured for ESX, QBCore, and Qbox servers.

## ESX

Select ESX as the framework in `config.lua` and make sure the ESX resource starts before Deez Chat.

## QBCore

Select QBCore as the framework in `config.lua` and make sure `qb-core` starts before Deez Chat.

## Qbox

Select Qbox as the framework in `config.lua` and make sure `qbx_core` starts before Deez Chat.

If your setup uses Qbox compatibility mode, verify that the compatibility bridge and all required Qbox dependencies start before Deez Chat.

For custom framework modifications, contact us through the Discord support channel.
