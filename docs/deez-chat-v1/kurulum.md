# Installation

1. Download the Deez Chat V1 package from your Tebex account.
2. Extract the downloaded archive.
3. Move the script folder into your server's `resources` directory.
4. Select your framework in `config.lua`.
5. Add the following line to your `server.cfg`:

```cfg
ensure deez_chat
```

6. Stop any other chat resource to prevent conflicts.
7. Restart your server.

> If the packaged resource uses a different folder name, use that exact name in the `ensure` line.
