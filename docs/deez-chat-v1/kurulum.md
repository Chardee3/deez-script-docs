# Kurulum

1. Deez Chat V1 paketini Tebex hesabınızdan indirin.
2. İndirdiğiniz arşivi açın.
3. Script klasörünü sunucunuzun `resources` dizinine taşıyın.
4. `config.lua` dosyasından framework seçiminizi yapın.
5. `server.cfg` dosyanıza aşağıdaki satırı ekleyin:

```cfg
ensure deez_chat
```

6. Varsayılan başka bir chat kaynağı çalışıyorsa çakışmayı önlemek için durdurun.
7. Sunucunuzu yeniden başlatın.

> Gerçek resource klasör adı farklıysa `ensure` satırında pakette gelen klasör adını kullanın.
