# Config Ayarları

Tüm temel ayarlar resource içerisindeki `config.lua` dosyasından yönetilir.

Yayın öncesinde bu sayfaya Deez Chat V1 paketindeki gerçek config seçenekleri ve açıklamaları eklenecektir.

> Config dosyasındaki değişken adlarını değiştirmeyin. Yalnızca desteklenen değerleri düzenleyin.
